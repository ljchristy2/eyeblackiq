The Model
How EyeBlackIQ finds where the market is wrong
What EyeBlackIQ Does

Eye black eliminates glare. It sharpens focus under pressure and lets you see what others can't. That's exactly what this model does.

On any given day, there are 300+ NCAA baseball games being played. The market sets lines on all of them — but the market isn't always right. EyeBlackIQ processes every available game, runs each one through a multi-factor model, and identifies where the opening or closing line represents a mispricing. That mispricing is the edge. That edge is what we post.

We don't predict winners. We find where the market is wrong.

How It Works

The model evaluates each game across multiple weighted factors including pitching metrics, schedule-adjusted performance ratings, recent team form, ballpark context, and line movement. Each factor contributes to a composite score that determines whether a line represents positive expected value — and if so, how much.

Picks are generated only when a market line is available. EyeBlackIQ does not post phantom picks or retroactive predictions. If the model sees no edge in a game, nothing gets posted. Discipline is part of the model.

The full proprietary formula is not published. What is published — every single day — is the result. The record speaks for itself.

The Tier System

Every pick is assigned a tier based on model confidence and calculated edge size.

🔥
Conviction
Maximum edge detected. The model is locked in. This is where the strongest expected value lives and where unit allocation is highest. Conviction picks are tracked separately and form the core of the season record.
📈
Value
Clear edge is present. The line has soft spots the model has identified. Strong plays with real expected value — not maximum confidence, but not a guess either. Solid unit allocation.
👀
Lean
The model sees something. Expected value exists but confidence is moderate. Lower unit allocation. Worth watching. Not worth max bet.
Pick Of The Day

One pick. The highest-confidence, highest-edge play on the full card. POD comes exclusively from the CONVICTION tier — never forced, never manufactured.

On days where no CONVICTION pick exists, there is no POD. Scarcity is intentional. The POD record is tracked separately from the full card record. Over time, it becomes the most meaningful single metric EyeBlackIQ publishes.

Transparency

Every pick is posted with the live market line at time of posting. Every result is recorded with the final score. Every loss is posted with the same visibility as every win. The full record — wins, losses, pushes, net units, ROI — is updated daily and published here.
